# Show Test
### Description
Simple script for displaying gpl, err, out, and cmd test files for [CSCI515](http://www.ecst.csuchico.edu/~tyson/classes/515.s16/).
### Usage
`./st [extension(s)] [test number]`
### Examples
`./st out t001`

`./st gpl 010`

`./st err 42`

`./st cmd 5`

`./st gpl err cmd out 184`
